# Screenshot instructions

1. Install Node.js (>=16) and npm.
2. From the project root run:

   npm install

3. Start the local preview server (if not already running):

   python -m http.server 8000

4. Run the screenshot script:

   npm run screenshot

Screenshots will be created in the project root:
- `screenshot-homepage-1366x768.png`
- `screenshot-homepage-1200x900.png`
- `screenshot-homepage-mobile-390x844.png`
